## 组件抽离的原则

1. 可复用的一定要抽离
2. 可配置的项， 在组件内部一定要抽离
3. 一定要训练自己的前瞻性， 用前瞻的眼光去抽离组件
4. 只要有独立的功能性， 一定要抽离组件、
5. 组件集合一定要有结构
